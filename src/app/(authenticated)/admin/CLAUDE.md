# Admin 도메인 (관리자 기능)

## 개요

관리자(ADMIN) 전용 기능을 제공하는 도메인입니다.

### 핵심 기능
- 대시보드: 재무 지표 시각화
- 직원 관리: 초대, 계정 관리
- 미수금 관리: 미수금 추적 및 완납 처리

---

## 접근 권한

- Role이 `ADMIN`인 사용자만 접근 가능
- 미들웨어에서 `/admin/*` 경로 보호

---

## 주요 기능

### 1. 대시보드
- 월별/연별 매출, 지출, 순이익 표시
- 미수금 상위 5개 매장 노출
- 매출/지출 추이 그래프
- 엑셀 출력

### 2. 직원 관리
- 직원 목록 (이름, 권한, 등록 상태)
- 초대 URL 생성 및 복사
- 직원 추가/제거 (soft delete)

### 3. 미수금 관리
- 미수금 목록 (월별/일별)
- 완납 체크 기능

---

## 관련 페이지

- `/admin/dashboard` - 대시보드
- `/admin/staff` - 직원 관리
- `/admin/outstanding` - 미수금 관리
