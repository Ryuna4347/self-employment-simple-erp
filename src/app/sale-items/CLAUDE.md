# SaleItem 도메인 (글로벌 품목 사전)

## 개요

시스템 전체에서 공유되는 품목의 원천 정보(마스터 데이터)를 관리하는 도메인입니다.

### 핵심 기능
- 글로벌 품목 마스터 데이터 관리
- 품목 검색 및 자동완성
- 매장/근무기록에서 품목 선택 시 참조

---

## 데이터 모델

### SaleItem
- `name`: 품목명 (규격 포함, 예: "콜라 500ml")
- `unitPrice`: 표준 권장 단가

---

## 비즈니스 규칙

### 마스터 데이터 역할
- SaleItem은 가이드 역할
- StoreItem, RecordItem은 값을 복사하여 독립 저장
- SaleItem 수정/삭제해도 기존 기록에 영향 없음

### 품목 입력 플로우
1. 품명 입력 시 Autocomplete로 검색
2. 선택 시 unitPrice 자동 입력
3. 직접 입력도 허용 (단가 수동 입력)

---

## 관련 페이지

- `/sale-items` - 품목 사전 관리 (Admin)
- 매장 관리의 품목 Autocomplete
- 근무 모달의 품목 Autocomplete
